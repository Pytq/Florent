Description

Cette extension permet d'écrire du texte avec rotation et déformation (comme pour l'italique).

TextWithDirection(float x, float y, string txt [, string direction])

x : abscisse
y : ordonnée
txt : texte
direction : une des valeurs suivantes (R par défaut) :
  • R : vers la droite
  • U : vers le haut
  • D : vers le bas
  • L : vers la gauche
TextWithRotation(float x, float y, string txt, float txt_angle [, float font_angle])

x : abscisse
y : ordonnée
txt : texte
txt_angle : angle du texte
font_angle : angle de déformation (0 par défaut)
